VM API v1

This is the default page of my VM API, which has a base URL of

http://home.bjd2385.com/api

At this time, valid endpoints include

  • api/list with required argument(s)

    • hosts (list)

    and optional argument(s)

    • status=[active|inactive]

    Example:

    (flask_vm_api) brandon@b350-gaming-pc:~/projects/flask_vm_api$ curl -X POST http://10.8.0.42:5000/api/list -H "Content-Type: application/json" -d '{"hosts":["perchost.bjd2385.com","b350-gaming-pc.bjd2385.com","routerhost.bjd2385.com"],"status":"active"}'
